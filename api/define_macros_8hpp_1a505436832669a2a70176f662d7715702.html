ASSIGNMENT(op)¶

provides converting binary operators, where one argument is of another type and has to be casted appropriately.

Parameters

  • Composed: final composed type provides converting binary operators, where one argument is of another type and has to be casted appropriately. ‘Composed’ used as default final composed type
  • op: operator provides assignment operators in form of +=, <<=, etc…
