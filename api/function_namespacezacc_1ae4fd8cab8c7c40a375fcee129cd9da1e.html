template <class InputIt>
std::string zacc::join(InputIt first, InputIt last, const std::string &separator)

Concatenates all the elements of a collection using the specified separator between each element. The collection value type must provide a str() method.

Return
A string that consists of the elements in provided collection delimited by the separator string
Template Parameters
  • InputIt: collection iterator type
Parameters
  • first: Begin
  • last: End
  • separator: The string to use as separator
